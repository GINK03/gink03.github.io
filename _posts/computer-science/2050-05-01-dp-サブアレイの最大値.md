# dp-サブアレイの最大値について

## 概要
 - `[x0, x1, ..., xn]`があるときに、どのindexでサブアレイを切り取ると、最大となるか
 - 累積和だけで対応できないようなケース
   - 最大となるには累積途中のマイナスを含んではならない
   - 動作をシミュレーションするために、dpが必要

## 具体例

```python
def solve(num):
    if len(num) == 1:
        return num[0]
    dp = [-float('inf')]*len(num)
    dp[0] = num[0]

    for i in range(1, len(num)):
        if dp[i-1] >= 0:
            dp[i] = num[i] + dp[i-1]
        else:
            dp[i] = num[i]
    return max(dp)

solve(num=[-2,1,-3,4,-1,2,1,-5,4]) == 6
```
