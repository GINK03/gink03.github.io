# アレイデータ構造について
いくつか有名なアレイデータ構造が存在し、各データ構造にはメリットとデメリットが存在する

 - ***linked list***
   - データ構造をポインタでつなぐデータ構造
 - ***array***
   - プリミティブな構造でできることが少ない
 - ***balanced tree***
   - 様々な条件で低い計算コスト`O(log n)`
 - ***hashed array tree***
