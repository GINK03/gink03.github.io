### 例; 偶数奇数で結果を判断できる

**問題**  
 - [CADDi 2018; D - Harlequin](https://atcoder.jp/contests/caddi2018/tasks/caddi2018_b)

**解答**  

```python
N = int(input())
A = [int(input()) for n in range(N)]

if all(a%2 == 0 for a in A):
    print("second")
else:
    print("first")
```
