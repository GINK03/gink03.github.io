# 最大公約数(greatest common divisor)について

## naiveは遅い

```python
from contexttimer import Timer

with Timer() as t:
    a, b = 3918848, 1653264

    for c in range(a+b, 0, -1):
        if a%c == b%c == 0:
            print(c)
            break
print(t.elapsed)

>> 3.46
```

## 早いアルゴリズム

```python
from contexttimer import Timer

with Timer() as t:
    a, b = 3918848, 1653264
    while b:
        a, b = b, a%b
    print(a)


print(t.elapsed)

>> 5e-05
```

## ライブラリ

```python
from contexttimer import Timer
import math

with Timer() as t:
    a, b = 3918848, 1653264
    print(math.gcd(a, b))


print(t.elapsed)

>> 5e-05
```
