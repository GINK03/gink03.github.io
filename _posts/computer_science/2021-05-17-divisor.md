# 約数列挙について
テンプレートになっている  
`10^12`ぐらいまで行ける  

## python

```python
def make_divisors(n):
    divisors = []
    for i in range(1, int(n**0.5)+1):
        if n % i == 0:
            divisors.append(i)
            if i != n // i:
                divisors.append(n//i)

    divisors.sort()
    return divisors
```
