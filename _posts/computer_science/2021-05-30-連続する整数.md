# 連続する整数
 - 尺取法を改良して`O(2N)`で計算できる
 
## colab
 - [colab](https://colab.research.google.com/drive/170n5TVnf87swuN56ZtqPtSxbOqlDRr-M?usp=sharing)

## 実装

```python
A = [2,3,4, 6,7, 9, 11,12, 15] # 5個のグループ

cnt = 0
i, j = 0, 0
while i < len(A)-1 and j < len(A)-1: 
    if i == j:
        i += 1
        cnt += 1
    elif A[i] + 1 == A[i+1]:
        i += 1
    elif A[i] + 1 != A[i+1]:
        j += 1
  
print(cnt) # 5
```
