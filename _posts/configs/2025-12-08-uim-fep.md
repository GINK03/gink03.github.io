# uim-fepの設定と使い方

## 概要
 - uim-fepはlinuxのターミナル上で動作する日本語入力フレームワーク
 - `mozc` と連携することで日本語入力が可能
 - 起動時に `uim-fep` コマンドを実行して使用
   - すべての入力が uim-fep を経由するので一部のキーが吸い込まれることがある

## インストール

**debian/Ubuntu系**

```bash
$ sudo apt install uim-fep uim-mozc
```

## 設定
 - `~/.uim` に設定を行う

**設定例**
```
;; デフォルトの入力方式をMozcに設定
(define default-im-name 'mozc)

;; Alt+` でオン／オフ（トグル）
(define-key generic-on-key?  '("<Alt>`"))
(define-key generic-off-key? '("<Alt>`" "escape" "<Control>c"))
```

 - alt+` でuim-fepのオンオフを切り替えられるように設定

## 起動例

```bash
$ uim-fep
```
