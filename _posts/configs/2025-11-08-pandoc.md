# pandocの使い方

## 概要
 - ドキュメント変換ツール
 - AIエージェントで作成したドキュメントのフォーマット変換に便利
 - haskellで実装されている

## インストール

**macOS**
```bash
$ brew install pandoc
$ brew install --cask basictex
```

## markdownからhtmlへの変換

**基本**
```bash
$ pandoc doc.md -o doc.html
```

**スタイルの埋め込み&幅の指定**
```bash
$ pandoc doc.md -s -o doc.html -V "maxwidth:70ch"
```
