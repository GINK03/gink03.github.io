# bunの使い方

## 概要
 - bunはTypeScriptの実行環境で、Node.jsの代替として使える
 - パッケージマネージャーとしても使用でき、`~/.bun` で管理されるのでシステムに影響を与えない
   - `~/.bun/bin` にバイナリがインストールされる

## インストール

```console
$ curl -fsSL https://bun.sh/install | bash
```

## パッケージのインストール

```console
$ bun install -g <package>
```

## パッケージの更新

```console
$ bun update -g
```
