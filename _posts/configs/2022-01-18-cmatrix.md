# cmatrixの使い方

## 概要
 - matrixという映画風のターミナルスクリーンセーバー
 - 派生が多数あり、日本語で同様のことを行えるjmatrixを作成した
 - 他にもターミナル上にキャンプファイヤーを表示するcacafireなどもある

## インストール
**ubuntu**  
```console
$ sudo apt install cmatrix caca-utils
```

## 実行

### cmatrix

```console
$ cmatrix
```

### cacafire

```console
$ cacafire
```

### jmatrix

```console
$ jmatrix.py
```
