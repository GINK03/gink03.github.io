# batの使い方

## 概要
 - a cat(1) clone with syntax highlighting and Git integration. 
 - 色味がきれいであり、lessやmoreの代わりのページャとしても使える

## インストール

**debian, ubuntu**  
```console
$ sudo apt install bat
```

**osx**  
```console
$ brew install bat
```

## オプション
 - `--style=plain`
   - 行数等の情報を表示しない
 - `--paging=always`
   - 常にページャー機能を有効化する 
 - `--theme=<theme>`
   - テーマを適応する

### 使用可能なテーマを表示する

```console
$ bat --list-themes
```
