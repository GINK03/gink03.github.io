# qrencodeの使い方

## インストール

**ubuntu**  
```console
$ sudo apt install qrencode
```

**osx**  
```console
$ brew install qrencode
```

## 文字列をQRコード化してターミナルに表示する

```console
$ echo "あいうえお" | qrencode -t utf8
```
