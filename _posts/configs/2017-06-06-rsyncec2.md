# Rsync EC2インスタンス
鍵越しでEC2インスタンスとrsyncとるの非常に面倒なので、一回,sshfsでマウントしてから、行う

## sshfs
```console
$ sshfs ubuntu@/home/ubuntu/{$SOURCR} {$TARGET} -o IndentityFile=~/td.pem
```

## rsync
```console
$ rsync -avL --progress {$SOURCE}/* ${TARGET}/
```
