# jqの使い方

## 概要
 - コマンドラインで動作するJSONパーサー
 - `.`が匿名の変数として使える
 - `.[]`が配列の展開となる
 - `|`でパイプライン

## インストール

**ubuntu/debian**
```console
$ sudo apt install jq
```

**macOS**
```console
$ brew install jq
```

## 具体例

### pretty print

```console
$ cat sample.json | jq .
```

### 配列の要素を選択

```console
$ echo '{"name": "John", "age": 30}' | jq '.name'
# "John"
```

### 配列の要素を選択

```console
$ echo '[{"name": "John", "age": 30}, {"name": "Mike", "age": 20}]' | jq '.[0].name'
# "John"
```

### 配列やオブジェクトをループして処理

```console
$ echo '[{"name": "John", "age": 30}, {"name": "Mike", "age": 20}]' | jq '.[] | .name'
# "John"
# "Mike"
```

### フィルターを使って複雑な条件を指定

```console
$ echo '[{"name": "John", "age": 30}, {"name": "Mike", "age": 20}]' | jq '.[] | select(.age > 25) | .name'
# "John"
```
