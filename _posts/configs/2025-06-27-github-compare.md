# github compareの使い方

## 概要
 - ブランチ間で差分を確認したいときの機能

## 使い方
 - 以下のURLにアクセスすることで、指定したブランチ間の差分を確認できる

```bash
https://github.com/<user-or-org>/<project>/compare/<base-branch>...<compare-branch>
```
