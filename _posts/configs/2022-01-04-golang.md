# golangのセットアップと使い方

## 概要
 - 速度が必要なAPI開発等で人気がある言語
 - パッケージマネージャを使わなくても`go get`でソフトウェアがインストールでき、binary形式で配置できる

## 言語のインストールとセットアップ

**debian, ubuntu**  
```console
$ sudo apt install golang
```

**osx**  
```console
$ brew install golang
```

### 言語の環境変数

```console
$ go env
```

### GOPATH環境変数
 - キャッシュやインストールされたバイナリの配置場所
 - デフォルトが"~/go"で邪魔であり、変更推奨

**変更例**  
```console
export GOPATH=$HOME/.go
```

### binaryの場所

`$GOPATH/bin`である
