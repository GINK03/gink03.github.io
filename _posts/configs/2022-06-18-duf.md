# dufの使い方

## 概要
 - dfをよりリッチにしたようなツール
 - ネットワークマウントしたデバイスやfuseでマウントしたデバイスを例外で弾いてくれるので便利

## インストール

**osx**  
```console
$ brew install duf
```

**debian, ubuntu**  
```console
$ sudo apt install duf
```

## 使用例

```console
$ duf
```
