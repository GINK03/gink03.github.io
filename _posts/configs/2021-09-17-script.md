# script(証跡ツール)の概要と使い方

## 概要

シェルにて`script`を実行すると、その後のstdoutに出力された内容を記録する  

## 使い方

**記録開始**  

```console
$ script <savefile-name>
```

**記録停止**  

```console
$ exit
```
