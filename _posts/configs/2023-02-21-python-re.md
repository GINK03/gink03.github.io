# pythonのreの使い方

## 概要
 - デフォルトライブラリに含まれる正規表現関連のライブラリ
 - prceを厳密に満たしておらず、より高機能な[/python-regex/](/python-regex/)も一応ある

## よく使うスニペット

### URLとマッチする正規表現

```python
url_pattern = r"(http|ftp|https):\/\/([\w\-_]+(?:(?:\.[\w\-_]+)+))([\w\-\.,@?^=%&:/~\+#]*[\w\-\@?^=%&/~\+#])?"
```
