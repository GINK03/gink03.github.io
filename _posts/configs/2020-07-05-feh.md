# feh
terminal(LinuxのKonsoleなど)で画像を表示するときのソフト  

## how to use

```console
$ feh ${TARGET}.png
```

## feh over ssh

```console
$ ssh -X ${REMOTE}
```

もっと一般的には、 `.bashrc` or `.zshrc` にaliasを設定する

```console
alias ssh="ssh -X"
```
