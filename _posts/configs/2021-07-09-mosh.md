# moshについて
 - sshのUDPラッパー
 - 通信が遅い or 安定性が低い環境下で使用できる

## 使用

**最小の構成コマンド**  

```console
$ mosh user@server 
```

**サーバの実行ファイルを指定する(osxをホストとするとき)**  

```console
$ mosh user@server --server=BIN_PATH
```

`BIN_PATH`は以下のケースが多い  

```console
$ mosh user@server --server=/usr/local/bin/mosh-server
```

**プライベートキーを指定して接続**  

```console
$ mosh --ssh="ssh -i KEY_PATH" user@server
```
