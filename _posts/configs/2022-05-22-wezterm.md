# Wez's Terminal Emulator(wezterm)の使い方

## 概要
 - osx, linux, windowsで動作するクロスプラットフォームのterminal emulator
 - 設計がモダンで、neovimと同様にluaで設定を記述できる
 - true colorに対応している
 - wezという人が作っており、rustで書かれている

## インストール

**osx**  
```console
$ brew install wezterm
```

**windows**  
```console
> scoop install wezterm
```
