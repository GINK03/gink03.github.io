私の使い方は、引数でロジックを発火させるかどうかのパターンが多いので、無理くり、clickを使うとこうなります。  

```python
import click


@click.command()
@click.option('--mode', default='make_sparse', help='mode spec')
def main(mode):
  if mode == 'make_sparse':
    make_sparse()

if __name__ == '__main__':
  main()
```
