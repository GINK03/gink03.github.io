# QR分解について

## 定義

正方行列$$A$$を直交行列$$Q$$と上三角行列$$R$$を用いて

$$
A = QR
$$

となっている行列のこと

$$Q$$はユニタリ行列である


## pythonでの実装

```python
from numpy.linalg import qr
import numpy as np

a = np.random.randn(9, 6)

q, r = np.linalg.qr(a)

np.allclose(a, np.dot(q, r)) # True
```
