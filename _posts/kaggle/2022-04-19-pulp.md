# pulpの使い方

## 概要
 - 線形最適化ツール
 - 様々な最適化を行える

## 具体例

```python
x = LpVariable("x", 0, 3)
y = LpVariable("y", 0, 1)

prob = LpProblem("myProblem", LpMinimize)

# 制約の追加
prob += x + y <= 2

# 目的関数の追加
prob += -4*x + y

# 最適化
status = prob.solve()
```

```python
display(value(x)) # 2.0
display(value(y)) # 0.0
```
