# 固有値について

## 概要

行列$$A$$で以下を満たす$$\lambda$$

$$
\det(A - \lambda I)= 0
$$

## simpyでの求め方

```python
from sympy import *
m = Matrix([[0, 1-I, 1], [1+I, 0, 1-I], [1, 1+I, 0]])
display(m)
display(m.eigenvals())
```

$$
