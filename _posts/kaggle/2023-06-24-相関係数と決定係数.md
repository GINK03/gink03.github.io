# 相関係数と決定係数

## 概要
 - excelでは相関分析では、決定係数(R^2)がよく出る
 - 相関係数(R)と決定係数(R^2)の関係はそのまま、相関係数の二乗が決定係数である
   - 相関係数と決定係数をごちゃごちゃに考えていると傾きのイメージがずれるので注意する

## Google Sheetsでの計算

```
=correl(B2:B, C2:C) // 相関係数
=correl(B2:B, C2:C)^2 // 決定係数
```

### Google Sheets
 - [correl example](https://docs.google.com/spreadsheets/d/1K2TEWEOFHb729zaFmI4wwiSW8eJnZ61lbjkTJBwHKC4/edit#gid=0)

## pythonでの計算

```python
from scipy.stats import pearsonr

rho, p = pearsonr(x, y)
print(f"相関係数 = {rho}, 決定係数 = {rho**2}")
print(f"p値 = {p}")
```
