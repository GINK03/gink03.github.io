# davinci resolve

## platforms
 - linux
 - windows
 - osx

## change language
 - [Davinci Resolve] -> [Preferenses] -> [General] -> [User] -> [Select language]

## add video, photo and sound to media pool
 - (カットの下のボタンをクリック) -> (左上のmedia poolにコンテンツをD&D) -> (エディットのタイムラインに追加できる)

## volume control

**微妙にわかりにくい**  
 - (♬マークをクリック) -> (ミキサーの幅を広げる) -> (横のウィールを伸ばして対象のソースを探す) -> (dBをいじる)
